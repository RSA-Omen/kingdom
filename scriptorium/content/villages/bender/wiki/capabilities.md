# Capabilities

_Stub — to be expanded as scripts are added._

**What Bender can do today**

- Open Pronto Xi screens under named credential profiles
- Execute pre-recorded keystroke sequences
- Capture screen output and return it as text or structured JSON

**What Bender cannot do (yet)**

- Anything Pronto cannot do via its UI
- Run two scripts in the same session concurrently
- Recover automatically from unexpected modal dialogs
