# Bender

A keystroke-level automation harness for Pronto Xi. Bender lets other systems trigger actions inside Pronto without an API — it drives the UI by sending keystrokes and screen-scrapes the result.

Bender is how the Kingdom talks to Pronto Xi for everything beyond URL routing (which is [[the-interceptor]]'s job).

See [[bender/sequence]] for how a typical call looks and [[bender/capabilities]] for what Bender can and cannot do today.
