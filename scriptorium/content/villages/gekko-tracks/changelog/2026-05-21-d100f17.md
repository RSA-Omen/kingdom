# fix(cardholder): close rejected-batch UX gaps + restore Zac tx #4320

Backfilled checkpoint covering three cardholder-batch fixes shipped 2026-05-21, all triggered by the Zac/$17.10 incident.

## 1. Drop cardholder Remove-from-batch capability (PR #17, merge `e874173`)

Cardholders had a "Remove" button on each batch row that called `DELETE /api/cardholders/{cid}/batches/{bid}/transactions/{tid}`. It orphaned the classification (`batch_id=NULL`, `status=unclassified`) and silently dropped the line from the batch — **including batches already in `in_review` with a manager**. Once removed, the row escaped the manager's rejection sweep (`managers.py:691-696`, which only updates rows where `batch_id` matches), so it landed in unclassified limbo with no rejection reason. Cardholders should not mutate batch membership at all; adding/removing lines is a Finance/Manager concern. Endpoint and frontend buttons deleted end-to-end.

## 2. Surface rejected batches in the inbox (PR #17, same merge)

The inbox filter only admitted `open` + `in_review`, so a rejected batch dropped into the collapsed "Show history" section. Cardholders got no signal a batch had been bounced back. Filter now includes `rejected`; history counter adjusted to compensate.

## 3. Allow re-submitting a rejected batch (PR #18, merge `d100f17`)

The submit endpoint's status guard only accepted `open` + `in_review`, so a cardholder who fixed a rejected batch and clicked Submit got a 400 from the backend even though the frontend enabled the button. Guard now includes `rejected`; the rejection → fix → resubmit loop works end-to-end.

## Live data fix (out of band)

Restored transaction #4320 ($17.10, Hayes Engineering, 15 Apr) to batch 126 with `status='rejected'` to match its 7 siblings. DB backup: `backend/ccc.db.backup-pre-tx4320-restore-20260521-030620`. A second backup `ccc.db.backup-pre-rollback-20260521-033111` was taken when a verification probe inadvertently submitted the batch — that was rolled back; Zac's in-progress re-classifications (1 confirmed, 7 rejected) were preserved.
