# fix(batches): partial submission sub-batch now inherits finance batch as parent

When a cardholder does a partial submission, the created sub-batch was pointing
to the cardholder batch (batch_id) as parent instead of the finance batch
(batch.parent_batch_id). The batch manager queries children of finance batches,
so sub-batches with a cardholder-batch parent were completely invisible to Finance.

Also fixed existing batch 133 (Tim Bell partial submission, 32 May txns) in the
live DB by updating parent_batch_id from 132 (cardholder) to 129 (finance batch).
